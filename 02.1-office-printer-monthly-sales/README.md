# 06 - Office Printer Monthly Sales

## Objective

Integrate Oracle APEX Office Print (AOP) into the sales app to generate printable monthly sales output including charts.

## Scope

- Configure AOP in environment/app
- Build monthly sales dataset package/query
- Produce report with chart(s), KPIs, and summary tables
- Support PDF output (optional Excel output profile)

## Dependencies

- Weekly sales upload data must already be processed
- Authorization must protect report access

## Deliverables

- AOP data source query/package
- Print template (Word/Excel-based)
- Printable monthly sales report page/action
