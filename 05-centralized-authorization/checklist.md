# Checklist

## Build Checklist

- [ ] Object catalog is populated from APEX metadata views
- [ ] Roles can be created and maintained
- [ ] Objects can be assigned to roles
- [ ] Users can be assigned to roles
- [ ] Central authorization function is reusable
- [ ] Menu/page/process checks use centralized authorization

## Validation Checklist

- [ ] Unauthorized users cannot access protected components
- [ ] Role updates take effect on next request/session
- [ ] Access reports show accurate effective permissions
- [ ] Admin-only functions are restricted correctly

## Done Criteria

- [ ] Access control is fully metadata-driven
- [ ] Permission changes no longer require code deployment
- [ ] Audit trail covers key authorization changes
